DUNGEON EXPLORATION

URL

https://github.com/chonomikiya/UnityDungeonExploration

対応プラットフォーム

Windows / MacOS / Linux

制作期間

3ヶ月

説明

Unityで作成したランダム迷路生成のゲームです。
ランダム生成のダンジョンに興味があり、またオブジェクト指向の振る舞いについていい練習になると思ったので挑戦しました。
穴掘り法の深さ優先のアルゴリズムを採用しており、偶に薄い壁をぶち抜きます。
アイテムのデータはSQLiteで管理しており、アイテムの追加はデータベースの更新のみで行えます。

反省点です。
まず作品のメインになる迷図生成のアルゴリズムに関して研究不足でした。
もっと調べておけば生成するダンジョンの幅を広げる事ができたと思います。
クラス設計がいい加減で、参照の参照なんか多く作ってしまった所も後悔しています。
